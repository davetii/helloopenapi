# Hello Open API SpringBoot 

Examples of using OpenApi in Springboot

## Description

This is the second sample which expands on hello world API using webmvc and includes basic model examples

### Dependencies
* Springboot 3 web mvc
* openapi libraries
* jdk17

### Installing
* mvn install

### Executing program
* mvn spring-boot:run
